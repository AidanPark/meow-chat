## 📋 Executive Summary

**냥닥터**는 고양이 건강검진 결과지를 AI로 분석하고, 반려묘의 전 생애주기에 걸쳐 맞춤형 건강 상담을 제공하는 **디지털 주치의 서비스**입니다.

> 🎯 **"숫자가 아닌, 이해되는 이야기로"**  
> 복잡한 검사 결과를 보호자가 이해할 수 있는 언어로 전달합니다.

| 핵심 지표 | 수치 |
|----------|------|
| 국내 반려묘 가구 | 약 300만 가구 |
| 펫 헬스케어 시장 규모 | 1.5조원 (2024) |
| 연평균 성장률 (CAGR) | 15%+ |
| 고양이 1마리 연간 의료비 | 평균 50만원 |

### 보호자의 Pain Point

| 문제 | 설명 |

### 사용자 시나리오

#### 시나리오 1: 건강검진 결과 분석

```
보호자: [혈액검사 결과지 사진 업로드]

냥닥터: 검사 결과를 분석했습니다! 📋

        ✅ 정상 범위: 적혈구, 백혈구, 혈소판
        ⚠️ 주의 필요: 크레아티닌 2.1 (정상 0.8-2.0)
        
        7살 코리안숏헤어 중성화 수컷 기준으로 보면,
        크레아티닌이 상위 15% 수준이에요.
        
        6개월 전 검사(1.8)와 비교하면 약간 상승했네요.
        신장 건강을 위해 수분 섭취량을 체크해보시고,
        한 달 후 재검사를 권장드려요.
        
        💧 급수량 늘리는 팁을 알려드릴까요?

보호자: 응 알려줘

냥닥터: 고양이 음수량 늘리는 방법이에요! 🐱💧
        
        1. 급수대 위치를 여러 곳에 배치
        2. 흐르는 물 급수기 사용 (고양이가 좋아해요)
        3. 습식 사료 비율 늘리기
        4. ...
```

#### 시나리오 2: 증상 상담

```
보호자: 우리 고양이가 오늘 밥을 안 먹어요

냥닥터: 걱정되시겠어요 😿 
        몇 가지 여쭤볼게요.
        
        1. 밥을 안 먹은 지 얼마나 됐나요?

        3. 구토나 설사가 있었나요?
        4. 평소와 다른 행동(숨기, 과도한 그루밍 등)이 있나요?

보호자: 아침부터요. 물은 조금 마셨어요. 다른 건 없어요.

냥닥터: 다행히 급성 응급 상황은 아닌 것 같아요.
        
        하루 정도 관찰해보시되, 다음 증상이 나타나면
        병원에 가시는 게 좋아요:
        
        ⚠️ 24시간 이상 완전 절식
        ⚠️ 구토가 2회 이상 반복
        ⚠️ 무기력하게 숨어있음
        
        우리 나비(기록된 이름)가 지난달 예방접종 후에도
        하루 정도 식욕이 떨어졌었죠? 
        스트레스 요인이 있었는지도 생각해보세요.
```

### 수익원 상세

| 수익원 | 설명 | 예상 객단가 |
|--------|------|------------|
| **프리미엄 구독** | 무제한 분석 + 상세 리포트 + 수의사 연결 | 월 9,900원 |
| **커머스 연동** | 건강 상태 맞춤 사료/영양제 추천 수수료 | 거래액 5~10% |
| **펫보험 제휴** | 건강 데이터 기반 보험 상품 연계 | 가입당 CPA |
| **동물병원 B2B** | 병원용 환자 관리 솔루션 | 월 10~50만원 |
| **데이터 인사이트** | 익명화 건강 데이터 연구/산업 협력 | 건당 협의 |

```
           - 습식/건식 선택 가능

        💡 참고: 처방식 변경은 수의사 상담 후 권장드려요.
```
