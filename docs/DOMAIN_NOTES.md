# 도메인 참고 (고양이 건강검진)
> 이 기준은 참고용이며, 이상 징후가 있으면 즉시 동물병원에 문의하는 것을 우선합니다.

- 중독 의심 물질 섭취
- 외상 또는 골절 의심
- 의식 저하 또는 경련
- 호흡 곤란 또는 개구 호흡
- 혈뇨 또는 배뇨 곤란
- 반복적 구토(2회 이상/일)
- 24시간 이상 완전 절식

## 3) 응급 상황 안내 기준(예시)

- 노묘(7세+): 연 2회 검진, CKD/갑상선, 관절/활동량, 종양 모니터링
- 성묘(1-7세): 연 1회 검진, 치과 건강, 비만 예방
- 키튼(0-1세): 예방접종, 중성화 시기, 성장 모니터링, 기생충 예방
