# 향후 확장 메모 (meow-chat)

> **목적**: 아이디어/후보 기술/확장 포인트를 자유롭게 기록합니다.
> `docs/plan.md`는 순수 계획만 유지합니다.

## 1) 제품/UX 확장

- 검사 결과 시각화(추세 그래프)
- 결과지 품질 체크(OCR 전처리 자동 튜닝 + 재촬영 가이드)
- 결과 비교(이전 검사 대비 변화)

## 2) 기술 확장

- RAG: 수의학 지식 베이스 + 출처 제시
- LangGraph: 단계형 워크플로우(분기/루프/검증)
- 메모리: 반려묘 프로필 + 건강 이력 장기 저장

## 3) 서빙/운영 확장

- Streamlit ↔ API(FastAPI) 분리
- 비동기 처리(큐/백그라운드 작업) 도입
- 관측 가능성(로그/트레이싱/비용) 강화

## 4) 실험 트랙

- 멀티 에이전트(AutoGen/CrewAI) PoC
- 코호트 기반 비교(나이/품종/성별/중성화)
